# social-media-app
social media application APIs using nodeJs and mongoDB

used with https://github.com/MA2592021/vbook1

## Steps to Run

- create .env file

  PORT = 5000
  
  TOKEN_KEY = lol
  
  DB_URL = mongodb://localhost:27017/testDB
  
- write in teminal:   
```
npm i
```

- then:   
```
npm run dev
```
